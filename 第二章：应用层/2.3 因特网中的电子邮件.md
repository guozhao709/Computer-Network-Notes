# 2.3 因特网中的电子邮件

三部分组成：用户代理、邮件服务器、简单邮件传输协议（SMTP）

邮件服务器是电子邮件体系结构核心，即是客户端也是服务端

SMTP使用的是TCP协议

用户代理 <--> 邮件服务器 <==========> 邮件服务器 <--> 用户代理

## 2.3.1 SMTP

是因特网电子邮件的核心

它限制了邮件报文只能采用简单的7bit ASCII码 表示

它不使用中间邮件服务器发送邮件

## 2.3.2 邮件报文格式

首部行 + 报文体

```
From: guozhao@qq.com
To: guozhaowqsa@xupt.edu
subject: Please put up homework.

------data------
```

## 2.3.3 邮件访问协议

IMAP, 一般用于邮件的接收, 允许用户管理自己的邮件服务器中的文件夹
