# 3.2 多路复用与多路分解

多路分解: 运输层的每个报文段都有几个字段, 标识出接受套接字, 根据这些字段将数据交付给正确的套接字

多路复用: 运输层从不同套接字接受报文, 加上首部字段(为了以后分解)封装成报文段, 再传递给网络层

运输层多路复用的要求:

1. 有唯一标识符
2. 每个报文段有特殊字段, 指示要接收到的套接字, 这些特殊的字段是源端口字段和目的端口字段

一个 UDP 套接字是由一个二元组全面标识的, (目的 IP 地址, 目的端口号)

如果源 IP 不同, 但目的 IP 和目的端口相同, 则经过相同的套接字

一个 TCP 套接字是四元组唯一标识的, (源 IP 地址, 源端口号, 目的 IP 地址, 目的端口号)

如果源不同, 即使目的相同, 也还是不同的套接字, 除非是报文段携带了初始创建连接的请求
