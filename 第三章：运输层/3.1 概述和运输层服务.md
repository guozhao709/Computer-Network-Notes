# 3.1 概述和运输层服务

运输层协议提供了进程间的逻辑通信(好像忽略了物理结构)功能

网络路由器仅作用于数据报的网络层字段

## 3.1.1 运输层和网络层的关系

网络层提供了主机间逻辑通信

运输层提供了进程间的逻辑通信

运输层的某些"保证"依赖网络层, 如提供时延或带宽保证

有些不需要, 如网络层可能会使分组丢失, 篡改和冗余, 但应用层可以提供可靠的数据传输服务

## 3.1.2 因特网运输层概述

因特网提供了两种协议:

- 面向连接, 提供可靠数据传输服务的 TCP

- 不面向连接, 不提供可靠数据传输服务的 UDP

IP 的服务模型是尽力而为交付服务, 不可靠服务(网络层)

将主机间交付扩展到进程间交付被称为运输层的多路复用与多路分解

数据交付和差错检查为最低限度的运输层服务
